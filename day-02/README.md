# 🩺 Day 2 - Medical Symptom Checker

## 🚀 Project Overview

This is Day 2 of my 100 Days Medical AI Challenge.

I built a Medical Symptom Checker web application that allows users to select symptoms and receive possible health condition suggestions along with basic recommendations.

## ✨ Features

- Symptom Selection
- Health Condition Suggestions
- Severity Level Detection
- Personalized Recommendations
- Responsive Design
- Modern User Interface
- Fast Analysis

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 📚 What I Learned

- Python Functions
- Lists
- Dictionaries
- Conditional Statements
- Basic Healthcare Logic
- Problem Solving
